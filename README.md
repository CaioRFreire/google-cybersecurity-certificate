# Google Cybersecurity Activities

## Overview
This repository contains selected hands-on activities completed during the Google Cybersecurity Professional Certificate.

The activities focus on foundational cybersecurity concepts, practical security operations, risk analysis, incident response, Linux administration, SQL filtering, Python automation, and security auditing.

# Activities

## 1 - Professional Statement
Developed a professional statement to include as part of a cybersecurity portfolio, highlighting professional goals, interests, and foundational cybersecurity knowledge.

## 2 - Controls and Compliance Checklist
Conducted an internal security audit to evaluate security controls, identify risks and vulnerabilities, and help ensure business continuity and asset protection.

## 3 - Incident Report Analysis
Used the NIST Cybersecurity Framework to analyze and respond to a security incident, focusing on incident identification, response, and recovery processes.

## 4 - File Permissions in Linux
Managed and reviewed file permissions in a Linux environment to ensure proper access control and support organizational security requirements.

## 5 - Apply Filters to SQL Queries
Used SQL queries and filtering techniques to investigate security-related information and support security monitoring activities.

## 6 - Vulnerability Assessment Report
Conducted a vulnerability assessment for a small business environment by identifying security weaknesses, evaluating risks, and outlining remediation recommendations.

## 7 - Incident Handler's Journal
Developed an incident handler’s journal to document security incidents, investigation steps, observations, and response actions.

## 8 - Algorithm for File Updates in Python
Developed a Python-based solution to automate file update processes and manage employee access to restricted content in a healthcare environment.
